# Advent of Code Solutions

This repository contains my solutions for the [Advent of Code](https://adventofcode.com/) challenges.

## Structure

The repository is organized by year and day.

## Years

- [2025](./2025)
  - [Day 1](./2025/day1)

## Running the Solutions

Most solutions are written in Python. To run a solution, navigate to the corresponding directory and run the script.

Example:

```bash
cd 2025/day1
python main.py
```
